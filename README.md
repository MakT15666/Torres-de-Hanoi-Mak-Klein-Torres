# Torres de Hanoi - Simulador Algorítmico y Educativ
## 📋 Descripción del Proyecto
Este proyecto consiste en una aplicación web interactiva y responsiva diseñada para la enseñanza y comprensión del clásico acertijo matemático de las **Torres de Hanoi**. La plataforma combina una interfaz de usuario moderna con un motor de cálculo lógico para demostrar conceptos clave de la ingeniería de software, como la recursividad, las estructuras de datos tipo Pila (Stacks) y el análisis de complejidad computacional exponencial.

## 🎯 Tema Seleccionado
* **Concepto:** Las Torres de Hanoi (Lógica Matemática aplicada al Análisis y Desarrollo de Software).
* **Enfoque Pedagógico:** Demostración práctica del crecimiento exponencial ($M = 2^n - 1$) y optimización algorítmica.

## 🛠️ Herramientas Utilizadas
El proyecto ha sido desarrollado siguiendo estándares modernos de desarrollo web sin dependencias externas:
* **HTML5:** Estructuración semántica del contenido y accesibilidad.
* **CSS3:** Diseño responsivo basado en Grid y Flexbox, variables globales (Custom Properties) y animaciones fluidas utilizando la paleta de colores corporativa (#2563EB, #1E3A8A, #F59E0B).
* **JavaScript (ES6+):** Manipulación dinámica del DOM, validación de formularios y lógica de cálculo algorítmico en tiempo real.
* **Render:** Plataforma en la nube utilizada para el despliegue automático y hosting del entorno web.

## ⚡ Funcionalidad Principal
La aplicación incluye un **Simulador de Complejidad** interactivo que permite al usuario:
1. Configurar dinámicamente el número de discos (de 1 a 30).
2. Seleccionar la velocidad de simulación del procesador (movimientos por segundo).
3. Calcular en tiempo real el número exacto de movimientos óptimos obligatorios mediante la fórmula matemática.
4. Obtener una estimación legible del tiempo de ejecución (traducido automáticamente a segundos, minutos, horas o días según la escala del resultado).
5. Recibir alertas de advertencia por desbordamiento de pila o crecimiento exponencial crítico al superar los 25 discos.
